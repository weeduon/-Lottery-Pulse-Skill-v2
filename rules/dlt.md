# 中国体育彩票超级大乐透玩法摘要

## 基本规则

- 前区：01-35 中选 5 个。
- 后区：01-12 中选 2 个。
- 基本投注：2 元 / 注。
- 追加投注：每注加 1 元，只参与浮动奖。
- 开奖频率：每周一、三、六开奖，休市日除外。
- 支持：单式、复式、胆拖、多倍投注。

## 中奖条件

| 奖级 | 条件 |
|---|---|
| 一等奖 | 前区5 + 后区2 |
| 二等奖 | 前区5 + 后区1 |
| 三等奖 | 前区5 + 后区0，或前区4 + 后区2 |
| 四等奖 | 前区4 + 后区1 |
| 五等奖 | 前区4 + 后区0，或前区3 + 后区2 |
| 六等奖 | 前区3 + 后区1，或前区2 + 后区2 |
| 七等奖 | 前区3 + 后区0，或前区2 + 后区1，或前区1 + 后区2，或前区0 + 后区2 |

## 理论概率

总组合数：C(35,5) × C(12,2) = 21,425,712。

一等奖理论概率：1 / 21,425,712。

## 分析要点

- 大乐透是两个独立号码池的组合问题。
- 推荐必须分开分析前区与后区。
- 前区指标：和值、跨度、奇偶、大小、三区分布、连号、尾数。
- 后区指标：奇偶、大小、和值、连号、遗漏。
